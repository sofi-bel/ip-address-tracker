## Build Setup
Clone the repo

```
git clone https://github.com/sofi-bel/ip-address-tracker.git
```
Install dependencies
```
npm install
```
Server with hot reload at http://localhost:1234/
```
npm run start
```
Production Build
```
npm run build
```
